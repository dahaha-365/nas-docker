# Mihomo

功能强大的代理程序

## 环境变量

| 变量名称           | 默认值       | 说明               |
|----------------|-----------|------------------|
| SUBSCRIBE_URL  |           | 订阅链接             |
| SUBSCRIBE_CRON | * 6 * * * | 更新订阅 cron        |
| GEO_CRON       | 0 5 * * * | 更新 geo 资源文件 cron |
| HTTP_PORT      | 7890      | http 代理端口        |
| SOCKS_PORT     | 7891      | socks 代理端口       |
| MIXED_PORT     | 7892      | 混合代理端口           |
| REDIR_PORT     | 7893      | redir 代理端口       |
| TPROXY_PORT    | 7894      | tproxy 代理端口      |
| API_PORT       | 9090      | rest api 端口      |
